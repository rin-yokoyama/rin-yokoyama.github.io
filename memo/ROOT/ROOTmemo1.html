ROOT/特殊関数いろいろ

Memo-ROOTへ戻る

つかいかた

ROOTにはいろいろと特殊関数が用意されています。これらの多くは MathCore でなく、MathMore ライブラリが必要です。予め、

gSystem->Load("libMathMore");

を実行してロードしておきます。

使える関数の例

  • ROOT::Math::legendre(unsigned int l, double x) //ルジャンドル多項式
  • ROOT::Math::assoc_legendre(unsigned int l, unsigned int m, double x)　//ルジャンドル陪関数
  • ROOT::Math::sph_bessel(unsigned int l, double x)　//球ベッセル(第一種)関数
  • ROOT::Math::sph_neumann(unsigned int l, double x)　//球ノイマン(第二種ベッセル)関数
  • ROOT::Math::riemann_zeta(double x)　//リーマン・ゼータ関数

などなど...
