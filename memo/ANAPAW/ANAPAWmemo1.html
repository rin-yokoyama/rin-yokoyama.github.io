ANAPAW/PAW コマンド

覚えたコマンド一覧。少しずつ増やしていきたい。

ヒストグラム

  • set hcol [color]
    • ヒストグラムの表示色を変える。[color] は頭から順に、(塗りつぶすかどうか0 or 1)(枠線色)(塗りつぶし色)。例えば1204 なら枠線赤、塗りつぶし青。02 なら枠線赤、塗りつぶしなし。
    • 0:無, 1:黒, 2:赤, 3:緑, 4:青, 5:黄　このへんは ROOT と同じ。元の水色はたぶん12。
  • xblow, xyblow
    • 1D, 2D ヒストグラムの一部を切り取って新たなヒストグラムを作る。範囲はマウスクリックで選択。切り取りたいヒストグラムを ht してから使う。
  • blow [xmin] [xmax] [ymin] [ymax]
    • ヒストグラムの一部を切り取って新たなヒストグラムを作る。範囲は数値で指定。切り取りたいヒストグラムを ht してから使う。
  • hist/del [HID]
    • ヒストグラム [HID] を消す。ファイルからは消えない。
  • hist/plot [HID]([xmin]:[xmax],[ymin]:[ymax])
    • ヒストグラムの指定範囲を拡大してプロット。
  • hcut [HID] [cutfilename].cut [plot option]
    • xygate 用の cut ファイルを作る。[plot option] として colz などを付けないとデフォルトの表示に戻ってしまう。実行すると cut ID と comment を聞かれるので、入力してあとはマウスクリックで範囲選択。右クリックで終了。
  • hist/file [LUN] [filename].hbk ! [Option]
    • ヒストグラムを書き出すファイルを開く。[LUN]はこのファイルに充てる unit ID。0にすれば空いてる番号を使ってくれる。128まで。[Option]は n で新規作成、u で既にあるファイルに追加。
  • hrout [HID]
    • [HID] 番のヒストグラムを書きこむ。
  • close [LUN]
    • [LUN] 番のファイルを閉じる。
