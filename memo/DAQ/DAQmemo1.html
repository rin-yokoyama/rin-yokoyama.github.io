DAQ/DAQPCの準備

Memo-DAQへ戻る

2013年8月、フランス実験用にDAQPCを新たに用意した時のメモ

OS

  • Scientific Linux 6.3
  • kernel 2.6.32.279.el6.x86_64
パッケージは最小限でインストールしたので、以下では必要なパッケージを入れる手順も含んでいます。既に入っていればスルーで。

ハード

  • VMEコントローラ：VME 7807rc
  • LAN接続： [DAQPC]---[HUB]---[VMIVME(1G base)] (これだけ)

手順

  • ネットワーク設定
  • VMIVMEのMACアドレスを調べる。
    ディスプレイを接続し、1G baseのLANポートに接続した状態で電源を入れると表示される。
  • dhcpの設定
  • tftpの設定
  • nfsの設定
  • OS image の作成
  • babirlのインストール
  • cmdvmeの作成
  • kernel modulesの作成
  • ROOT&anarootのインストール
