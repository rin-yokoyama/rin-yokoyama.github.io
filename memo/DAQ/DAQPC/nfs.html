DAQ/DAQPCの準備/nfs設定

Memo-DAQ-DAQPCの準備へ戻る

　　　　　　　

概要

VMIVMEとDAQPC間のnfsマウントを設定する。

手順

  • yum install nfs-utils
  • ポート番号の固定
    • /etc/sysconfig/nfsを開く
    • 例えば以下のようにし、番号を確認する。（ここではデフォルトで書かれていた番号をそのまま使っています。）
      LOCKD_TCPPORT = 32803
      LOCKD_UDPPORT = 32769
      MOUNTED_PORT = 892
      STATD_PORT = 662
  • firewallの設定
    • /etc/sysconfig/iptablesを開く
    • 上で確認したポートと2049(nfsd),111(portmap)を解放
      -A INPUT -p udp -m state --state NEW -i eth0 -m udp -m multiport --dports 32769,892,662,2049,111 -j ACCEPT
      -A INPUT -p tcp -m state --state NEW -i eth0 -m tcp -m multiport --dports 32803,892,662,2049,111 -j ACCEPT
  • /etc/exports に以下を追加。
    • babirl の PATH を read only で
      /usr/babirl/bin 192.168.1.1/255.255.255.0(ro)
    • コントローラごとのconfigureをread/writeで
      /home/daq/daqconfig/CONTROLLERNAME 192.168.1.10(rw)
  • service nfs start
